# Mini Voting System in C

## Overview

This project is a simple console-based voting system developed in C. It allows users to cast votes for candidates and displays the final voting results.

## Features

- Cast votes for candidates
- Count total votes
- Display election results
- Menu-driven interface

## Technologies Used

- C Programming

## How to Run

1. Compile the program:
   ```bash
   gcc ic_code.c -o voting_system
   ```

2. Run the program:
   ```bash
   ./voting_system
   ```
